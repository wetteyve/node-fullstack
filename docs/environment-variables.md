# Environment Variables

- `RS911_CMS_API` - API base path to the CMS for the 911rs website.
- `RS911_CMS_KEY` - API token (read-access) to the 911rs CMS.
- `UHT_CMS_API` - API base path to the CMS for the uht-herisau website.
- `UHT_CMS_KEY` - API token (read-access) to the uht-herisau CMS.
- `UHT_CMS_SERVER_KEY` - API token (write-access) to the uht-herisau CMS.
- `UHT_DOWNLOAD_REGISTRATIONS_KEY` - Pass to access the registrations download.
- `RESEND_UHT` - Resend API token to send uht-herisau mails.
