# Development Environment Setup

## Prerequisites

- [Yarn](https://yarnpkg.com/en/)
- Node.js (check required version in [package.json](../package.json) `engines` field)

Tip: Use [nvm](https://github.com/creationix/nvm) to manage Node.js versions.

## Installation

```sh
yarn install
```

## Development Server

```sh
yarn dev
```

Available at [http://localhost:3000](http://localhost:3000).
